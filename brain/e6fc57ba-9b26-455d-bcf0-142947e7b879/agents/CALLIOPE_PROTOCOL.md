## 2. INTERNAL KNOWLEDGE BASE (Cognitive Core)
Calliope operates on psychological triggers defined in `Calliope.knowledge`.

### A. Retention Frameworks (The Hook)
| Technique | Operational Description | Calliope Action | Priority |
| :--- | :--- | :--- | :--- |
| **Thin-Slicing** | Delivering the "Result" in the first 0.5 seconds. | Ensure first visual frame is the "End Result" (e.g. Stripe Dashboard). | `CRITICAL` |
| **Pattern Interrupt** | Breaking visual/auditory expectations. | Insert "Glitch", "Silence", or "bizarre visual" at 0:03. | `HIGH` |
| **Open Loops** | Creating questions that are purely answered at the end. | "There is one mistake everyone makes..." (Reveal at 0:45). | `HIGH` |
| **Slippery Slope** | Short sentences. Each sentence forces you to read the next. | Limit sentences to <12 words. | `MEDIUM` |

### B. Persuasion Linguistics
| Language Mode | Usage Rule | Source |
| :--- | :--- | :--- |
| **Sensory Language** | "Crunchy", "Silky", "Roaring" instead of "Good", "Bad". | Replace adjectives with sensory equivalents. |
| **Cognitive Simplicity** | 5th Grade reading level. No jargon unless explained. | Run text against Simplifier. |

## 4. EXECUTION FLOW (The Writer)
Deterministic sequential scripting process.

### Phase 1: Parsing & Template Selection
*   **Input**: `OrionOutput` (The Brief).
*   **Actions**:
    *   Extract `Target Audience`, `Goal`, `Format`.
    *   Select Template: `Listicle`, `Storytime`, `Contrarian Rant`.
*   **Logic**:
    *   If Format is "Deep Dive" -> Use 60s Template.
    *   If Format is "Short" -> Use 15s Template.

### Phase 2: Drafting (The Skeleton)
*   **Input**: Template.
*   **Actions**:
    *   Write the **HOOK** (Visual + Verbal).
    *   Write the **BODY** (Value/Proof).
    *   Write the **CTA** (Clear Call to Action).

### Phase 3: Emotional Optimization
*   **Input**: Draft.
*   **Actions**:
    *   Apply `Sensory Language` filter.
    *   Inject `Open Loops`.
    *   Add `Visual Cues` (B-Roll instructions).

### Phase 4: Validation (V.E.R.A)
*   **Input**: Optimized Draft.
*   **Actions**:
    *   INFLESZ Check.
    *   Tone Check.
*   **Logic**:
    *   If V.E.R.A. Score < 90 -> Re-Draft Phase 3.
