## 2. WHY THIS DESIGN IS CORRECT (Validation)
Combining these roles into one agent creates two fatal flaws:
1.  **The "Hallucination" Trap**: If a writer agent needs a fact to make a sentence flow better, it often invents one. By separating them, Orion provides the *Hard Constraints* (Facts) that Calliope *cannot* alter, only decorate.
2.  **The "Boredom" Trap**: If a research agent writes scripts, it over-explains. Orion is *allowed* to be boring (it's a brief). Calliope is *mandated* to be exciting.

## 3. THE HANDSHAKE PROTOCOL
*   **Orion** says: *"The data proves 'Agentic AI' is trending ($25 RPM). The angle is 'Passive Income'."*
*   **Calliope** translates: *"[VISUAL: Money Printer] Stop trading time for money. Here is the AI stack that pays you while you sleep."*

**Conclusion**: The modular design is optimal for high-quality, scalable production.
