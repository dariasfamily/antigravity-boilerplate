# Guía de Configuración de Credenciales (Gemini Pro)

## 5. Gemini API Key

Para activar la "inteligencia central" (Gemini 2.0 Pro) y permitir el routing avanzado, necesitamos una API Key de Google AI Studio.

1.  Ve a **[aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)**.
2.  Haz clic en **"Create API key"**.
3.  Puedes crearla en un proyecto nuevo o uno existente de Google Cloud.
4.  Copia la clave (empieza por `AIza...`).
