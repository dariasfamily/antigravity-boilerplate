# Skill: Thumbnail Creation (Thalia)

**Role**: Production
**Input**: Title, Hook, Visual Concept
**Output**: Image File (.png/.jpg)

## Process
1.  **Conceptualize**: Create prompt based on "Curiosity Gap".
2.  **Compose**: Apply "Rule of Thirds" and Safe Zones.
3.  **Generate**: Call Image Gen Model.
4.  **Upscale**: Ensure high resolution.
