# Calliope Command Interface

Este comando activa el panel de control y estado de **CALLIOPE**.

// turbo
1. Cargar perfil desde `hive/agents/calliope/README.md`.
// turbo
2. Mostrar Misión, KPIs de Impacto Creativo y estado de la cola de producción.
// turbo
3. Ofrecer acceso rápido a workflows: `/calliope-create`, `/calliope-tone`.

**Uso:** `/calliope` para reporte de estado o solicita "Calliope, [tarea]".
