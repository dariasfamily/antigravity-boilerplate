# Egeria Command Interface

Este comando activa el panel de control y estado de **EGERIA**.

// turbo
1. Cargar perfil desde `hive/agents/egeria/config.json`.
// turbo
2. Mostrar Misión, KPIs actuales y estado de la Federación MCP.
// turbo
3. Listar opciones rápidas: `/egeria-sync`, `/egeria-audit`.

**Uso:** `/egeria` para reporte de estado o solicita "Egeria, [tarea]".
