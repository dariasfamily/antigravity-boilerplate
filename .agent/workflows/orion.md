# Orion Command Interface

Este comando activa el panel de control y estado de **ORION**.

// turbo
1. Cargar perfil desde `hive/agents/orion/README.md`.
// turbo
2. Mostrar Misión, KPIs de Impacto Estratégico y lista de Blueprints activos.
// turbo
3. Ofrecer acceso rápido a workflows: `/orion-plan`, `/orion-analyze`.

**Uso:** `/orion` para reporte de estado o solicita "Orion, [tarea]".
