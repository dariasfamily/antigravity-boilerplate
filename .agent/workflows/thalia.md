# Thalia Command Interface

Este comando activa el panel de control y estado de **THALIA**.

// turbo
1. Cargar perfil desde `hive/agents/thalia/config.json`.
// turbo
2. Mostrar Misión, KPIs actuales y estado de la Federación MCP.
// turbo
3. Listar opciones rápidas: `/thalia-sync`, `/thalia-audit`.

**Uso:** `/thalia` para reporte de estado o solicita "Thalia, [tarea]".
